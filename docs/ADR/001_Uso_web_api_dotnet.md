# ADR-001 - Uso do Asp.Net core Web Api com Controladores

## Contexto

O sistema é focado em desenvolver uma ferramenta útil para profissionais autônomos da psicologia, bem como ser um exercício de aprendizagem para o desenvolvimento fullstack com C# e .Net. 


## Decisão

Asp.Net core Web Api com Controladores foi escolhido por poder escalar com maior facilidade, ter estrutura bem delimitada e amplo uso profissional.

## Consequências

- Estrutura mais organizada
- Melhor escalabilidade
- Experiência mais próxima do mercado de desenvolvimento
